# Guía de Contribución 🤝

¡Gracias por interesarte en mejorar **Plataforma WhatsApp**! Para mantener la calidad del código y la organización, seguimos este flujo de trabajo:

## 🛠 Cómo contribuir

1. **Reportar un Error:** Abre un Issue detallando el problema y los pasos para reproducirlo.
2. **Sugerir una Mejora:** Abre un Issue describiendo la funcionalidad y por qué es útil.
3. **Enviar código:**
    - Haz un **Fork** del proyecto.
    - Crea una rama nueva `git checkout -b feature/nombre-de-tu-mejora`.
    - Realiza tus cambios y haz commit con mensajes claros.
    - Envía un **Pull Request** hacia la rama `main`.

## 📏 Estándares de Código
- Por favor, sigue las convenciones de Python (PEP 8).
- Incluye pruebas unitarias para cualquier nueva funcionalidad.
- Mantén la documentación actualizada si cambias la forma en que algo funciona.

---
*¡Cualquier duda, abre un issue y estaremos encantados de ayudarte!*
